# matematik-final-soru1
matematik final soru1
istanbul kent üniversitesi maatematik ödevi için c++ programa dili kullanarak geliştirilmiş ,ikinci dereceden fonksiyonların köklerini hesaplayan bir uygulamadır.
öğrenci bilgileri:
ad soyad: cud radvan 
öğrenci no: 2507020075
bölüm : bilgisayar mühendisliği 
programın çalışma mantığı : 
öncelikle kullanıcıdan ax^2+bx+c denklemindeki a b c değerlerini girmesini istedim bu değerlere göre programındeltay hesaplamasını sağladım
delta değerlerine üç olasılık tanıdım ya 0a eşit ya küçük ya da ikisi değildir yani 0dan büyüktür.
delta sıfırdan küçükse kök yoktur. eşit ya da büyükse de olan kökleri bulması için işlem atadım.
